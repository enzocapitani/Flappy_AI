# Flappy AI
Um projeto que recria o clássico jogo do Flappy Bird e insere uma rede neural para que ela aprenda a jogar o jogo através do algoritimo genético, foi feito usando somente Java e suas bibliotecas nativas.

## O que é um algoritimo genético?
O Algoritimo Genético é uma simulação da teoria da evolução porposta pelo biólogo Charles Darwin, no qual, no contexto do projeto, gera 100 pássaros por geração e separa os indivíduos que foram mais longe na rodada, quando todos perdem, separa os melhores indíviduos e faz uma espécie de "Cross Over" de pesos da rede neural e faz com que a próxima geração tenha essa mistura dos melhores "genes". Assim, em consequencia, a rede neural aprende a jogar.
